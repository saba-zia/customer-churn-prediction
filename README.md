# Customer Churn Prediction

## Project Overview
This project focuses on predicting customer churn using a real-world Telco customer dataset.

Customer churn means that a customer stops using a company’s service. The goal of this project is to understand customer behavior and build a machine learning model that can predict whether a customer is likely to churn.

## Dataset
The dataset used in this project is the Telco Customer Churn dataset from Kaggle.

Each row represents one customer. The dataset contains information such as customer demographics, services used, contract type, monthly charges, total charges, and whether the customer churned.

## Problem Statement
The main question of this project is:

Can we predict whether a customer will churn based on their account information and service usage?

## Project Goals
- Load and understand the dataset
- Perform exploratory data analysis
- Clean and preprocess the data
- Prepare features for machine learning
- Train classification models
- Evaluate model performance
- Document the full workflow clearly
